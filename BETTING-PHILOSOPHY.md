## 核心认知

**竞彩返奖率71%** = 每100元数学上只能收回71元。长期盈利需要持续比市场准确29%。

## 四大原则

### 1. 找偏差
P竞彩 = 0.71 ÷ 赔率。P真实 = (AI预测 + 球员评分 + 防守分析) ÷ 3。
P真实 > P竞彩 → 正EV → 投。
P真实 < P竞彩 → 负EV → 跳过。

### 2. 逆向市场
- 大众追热(葡1.13) → 押让球盘而不是胜平负
- 大众怕平(贵) → 在真·平局候选押平
- 大众追高赔冷门 → 不跟

### 3. 学会不投
- AI和竞彩方向矛盾 → 跳过或最多1注
- 赔率太低(<1.20) → 转投比分/让球
- 信息太少 → 跳过

### 4. 比分优先于胜平负
比分31选1 vs 胜平负3选1 → 比分竞争少、偏差大、正EV机会多。
