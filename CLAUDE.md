```
### 🧠 第二十层: 赔率认知 (每次必跑)

**核心不等式**: 竞彩返奖率71% → 需要持续比市场准确29%才不亏。赔率≠概率。

**操作**:
```
每场计算: P真实 vs P竞彩
P竞彩 = 0.71 ÷ 赔率
P真实 = (AI预测 + 球员评分 + 防守分析) ÷ 3

若 P真实 > P竞彩 → 正EV → 投
若 P真实 < P竞彩 → 负EV → 跳过或减注
```

**四大原则**:
| 原则 | 内容 |
|------|------|
| 找偏差 | P真实 > P竞彩=有价值，差距越大投越多 |
| 逆向市场 | 大众追热→押穿盘；大众怕平→押平；大众追冷→不跟 |
| 学会不投 | AI-竞彩矛盾、信息不足、赔率<1.20→跳过 |
| 比分优先 | 比分31选项→竞争少→偏差大→正EV比胜平负多 |
```

每场比赛取开球时辰排四柱：年柱(丙午)、月柱(戊午)、日柱(黄历查)、时柱(开球时辰)。
- 断日主旺衰：身旺则强队稳，身弱被克则冷门风险高
- 看刑冲：时柱冲日柱=比赛必有变故(补时绝平/红牌/逆转)
- 看用神：日主用神=克敌制胜的关键五行
- 🔮 15号验证：瑞士vs卡塔尔→庚申日寅时→庚金被午火克+寅木耗=身弱→瑞士虽强却赢不了。午午自刑=补时变故。**1:1精准命中**

### 🟣 第七层: 球员状态评分 — WebSearch + wc26 (赛前必跑)
每场比赛前，对两支球队分别跑100分制球员状态评分：

**评分模型 (4维度×100分)**

| 维度 | 满分 | 数据来源 | 评分规则 |
|------|:---:|------|------|
| 伤病影响 | 30分 | wc26 `get_injuries` + WebSearch | 全员健康=30；1核心缺=20；2核心缺=10；≥3核心缺=5 |
| 近期战绩 | 25分 | WebSearch "队名 热身赛 2026" | 5连胜=25；3胜1平=20；五五开=15；连败=5-10 |
| 球星卡位 | 25分 | wc26 `get_team_profile` + WebSearch 俱乐部数据 | 3核心全在巅峰=25；1核低迷=20；2核低迷=15；全萎=10 |
| 环境适应 | 20分 | wc26 venue + 海拔数据 + 天气预报 | 主场=20；中立=15；高原不适=10；含天气±5调节 |

**数据采集规则:**
- 先调 `wc26 get_team_profile` 取关键球员名单+伤病
- 再调 `wc26 get_injuries` 取伤病详情
- 最后 `WebSearch "<队名> 世界杯 热身赛 最新状态 伤病 2026年6月"` 取临场情报

**球员评分用于:**
- 修正AI胜率预测 (±5-10%)
- 调整投注方案中该队的比分选择方向
- 发现「市场高估/低估」的冷门信号
